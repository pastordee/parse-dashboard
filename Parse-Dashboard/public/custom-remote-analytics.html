üîß Remote Analytics Configuration

This dashboard connects directly to your analytics server for real-time data:

  • Primary Server: 192.168.0.138:${ENV.MAIN_PORT} (your remote server)
  • Fallback Server: localhost:3001 (local demo server)
  • Protocol: HTTP with CORS enabled
  • Update Interval: 30 seconds
  • Current Status: Loading...

Demo Server API Endpoints:

  • /api/analytics/overview - Overall metrics
  • /api/analytics/users - User statistics
  • /api/analytics/performance - Server performance
  • /api/analytics/realtime - Live data updates

How it works:

  1. First tries to connect to local demo server at localhost:3001
  2. If demo server fails, tries your remote server at 192.168.0.138
  3. If both fail, falls back to mock data for demonstration
  4. Demo server provides realistic, live-updating analytics data
